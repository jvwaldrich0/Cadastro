Com o propósito de melhorar minhas habilidades na linguagem Python, 
crei um software de cadastro utilizando pandas, hashlib, etc.

Explicação do começo:
-> No início você precisa digitar o nome de uma planilha e para isto você possui duas opções:
	1. Usar uma existente
		-> Para utilizar uma planilha já existente você pode escreve apenas 
		o nome de qualquer planilha
	que esteja na pasta 'Planilhas'
		-> Por padrão o software abre a planilha ENTRA21.csv ao apertar enter
	2.Criar uma planilha
		-> Se você quer criar planilha apenas escreva algum nome de planilha que 
		não exista na pasta 'PLanilhas'
	3.Observações
		-> Todos os nomes escritos serão formatados para caixa alta
